# vue3  林大实验室预约系统

  本项目为前端项目，主要基于TypeScript与vue 3.0.5框架实现,使用element-plus框架提供友好美观的互交操作。使用vuex管理全局数据状态, 引入axios-mock-adapter模拟后端数据,通过axios拦截器实现请求全局异常处理。token保存在SessionStorage防止页面刷新登录数据丢失,基于后端返回的角色权限动态生成路由信息。
